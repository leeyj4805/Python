# 정수형

* Integer은 정수를 다루는 자료형이며 `양의 정수,음의 정수,0`이 있다. 

### 1-1. 정수형
	a = 1000 # 양의 정수
	print(a)
	
	a = -7 # 음의 정수
	print(a)
	
	# 0
	a = 0
	print(a)
